Bridging the Gap

Fine-Tuning a Transformer Model for Slang & Informal Text Translation

The Challenge of Modern Language

Digital communication is filled with slang, abbreviations, and informalities that standard machine translation models struggle to understand. This project tackles this issue by fine-tuning a pre-trained neural network to accurately translate internet slang into formal language, making communication more accessible and universally understandable.

Project Lifecycle: From Data to Deployment

1. Data Collection

Gathered parallel informal/formal text pairs from online communities and manual annotation.

➤

2. Preprocessing

Cleaned, normalized, and tokenized text data to prepare it for the model.

➤

3. Model Fine-Tuning

Fine-tuned a T5-small model on the custom dataset for 5 epochs.

➤

4. Evaluation

Assessed model performance using BLEU scores and qualitative analysis.

Dataset Composition

Our training data was carefully curated from diverse sources to capture a wide range of informal language.

A balanced dataset ensures the model isn't biased towards one type of slang.

Why the T5 Model?

  • ✓

    Text-To-Text Framework

    T5 treats every NLP problem as a text-to-text conversion, making it perfectly suited for translation tasks.

  • ✓

    Pre-trained Knowledge

    It was pre-trained on a massive corpus (C4), giving it a strong foundational understanding of language.

  • ✓

    Scalability

    Available in various sizes (small, base, large), allowing for a trade-off between speed and performance.

Model Performance Over Time

The model's accuracy, measured by the BLEU score, improved significantly with each training epoch, indicating successful learning.

Translation in Action: Before & After

Example 1

Informal Input:

"idk, that new movie was kinda mid tbh"

Formal Output:

"I do not know, that new movie was somewhat mediocre to be honest."

Example 2

Informal Input:

"spill the tea, what happened at the meeting?"

Formal Output:

"Please share the details, what happened at the meeting?"

Example 3

Informal Input:

"we're gonna be late, we gotta dip asap"

Formal Output:

"We are going to be late, we have to leave as soon as possible."
